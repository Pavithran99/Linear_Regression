# Linear_Regression
Implement Simple Linear Regression Algorithm using numpy and see the output using matplotlib


### Dependencies:
1. Python - 3.7.6
2. numpy - 1.18.1
3. matplotlib - 3.1.2

### Steps To Follow:
1. Clone this repo
2. Run python lr.py
